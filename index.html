libflashfiler by chrisforbes

C Library providing direct access to TurboPower FlashFiler databases

Dependencies

None

Install

Compile with Visual C++ 8/9; Link against import library libflashfiler.lib and use the header libflashfiler.h; Deploy libflashfiler.dll with your application or in the PATH.
